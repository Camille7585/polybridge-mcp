## 🧭 What this app does

polybridge-mcp is a bridge hub for apps that work with AI tools. It helps you connect one language model to services like n8n, Blender, Notion, and other tools through one place.

This app is made for people who want one simple path instead of many separate setups. It gives you a single gateway that can pass requests between your AI tool and the service you want to use.

## 🔍 Troubleshooting

If the app does not open, try these steps:

- Download the file again

- Right-click the app and choose Run as administrator

If your AI app cannot connect:

- Make sure polybridge-mcp is still running

- Allow the app through Windows Firewall
- Close and reopen both apps
- Try the default settings first

If a tool does not appear in the hub:

- Confirm the tool is installed on your PC
- Check the tool name in the app settings
- Restart polybridge-mcp after changing settings
- Reconnect from your AI client
